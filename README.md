## This Repository to Built Backend App using Rust Axum Framework

1. hello-world: A simple Rust web server built with Axum framework demonstrating 5 different endpoints. show how to take query parameters, path variables, and JSON body in request and response.
2. crud-api-inmemory: A simple CRUD API built with Axum framework that uses an in-memory data store to manage items. It demonstrates how to create, read, update, and delete items using RESTful endpoints.
